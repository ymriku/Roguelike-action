# Game Design

## Concept

2Dローグライク横スクロールアクションRPG。短いステージを探索し、近接戦闘と回避で敵を倒しながらビルドを成長させる。ステージは毎回変化し、プレイヤーはランごとに装備、スキル、パッシブ効果を選んで攻略する。

## Core Loop

1. 拠点または開始地点からランを開始する。
2. 自動生成された横スクロールステージを進む。
3. 敵、罠、宝箱、イベントを処理する。
4. 戦闘報酬でスキルやステータスを強化する。
5. ボスまたは脱出口に到達して次の階層へ進む。
6. 失敗時は一部の恒久成長を保持して再挑戦する。

## Phase Goals

### Phase1

- プレイヤー移動
- ジャンプ
- 二段ジャンプ
- ダッシュ回避
- カメラ追従
- 仮プレイヤースプライト

### Phase2

- 基本攻撃
- ヒット判定
- 敵の仮実装
- HPとダメージ
- ノックバック

### Phase3

- ステージ生成
- 部屋、足場、接続通路
- 敵配置
- 報酬配置

### Phase4

- クラス、武器、スキル
- ローグライク報酬
- UI
- サウンド

## Controls

- A / D: 左右移動
- Space: ジャンプ、二段ジャンプ
- Shift: ダッシュ回避

## Design Pillars

- 操作は軽く、判断は重くする。
- 戦闘は回避、位置取り、攻撃タイミングを中心にする。
- ランごとの変化はステージ構造、敵編成、報酬選択で出す。
- 実装は小さい垂直スライスを積み上げる。
